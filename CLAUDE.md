# FinTrack

Personal finance dashboard with Django backend and React frontend.

## Backend
- Django 4.2 + DRF at backend/
- PostgreSQL database
- pandas for data processing

## Frontend
- React 18 + TypeScript at src/
- Redux Toolkit for state
- Recharts for visualization

## Running locally
docker-compose up -d
# Or manually:
cd backend && python manage.py runserver
npm run dev
